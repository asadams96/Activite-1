Si vous souhaitez lancer les fichier "Hello.java" et "Bye.java", vous devez posséder un IDE (comme Eclipse) ainsi qu'un JRE/JDK. Cependant rien ne vous empêche d'en lire le contenu dans votre éditeur de texte préféré.

Le fichier "Hello.java" affiche dans la console "Hello World !"
Le fichier "Bye.java" affiche dans la console "Bye World !"
